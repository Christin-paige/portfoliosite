A Case Study

Overview

IndieFilms is a client-side application written in JavaScript using React as a framework. It is connected to a movie database created in Mongo DB. IndieFilms delivers the user with a list of independent films and information about each film including an image of the movie poster, the summary, genre, and director. Users can create a profile, update information, and “favorite” individual films to store in a list on their profile page.

Purpose and Context

IndieFilms is a personal project I completed for my CareerFoundry Full-Stack Development course. The purpose of this task was to show my mastery of full-stack web development. I wanted to create
an app for independent film enthusiasts to have a niche website to keep track of their favorite films while having
the opportunity to build a full-stack web app from scratch.

Approach

Postman PUT request

MongoDB movie collection database

Server-Side

I created a REST API using Express and Node.js which interacts with a database built using MongoDB. The API provides information in JSON format and was tested in Postman. The API was then deployed to Heroku.

Client-Side

Based on the API and database, the user interface is a SPA, a single page application,
was built using React for styling and responsiveness, I utilized React Bootstrap.

Features Include

  • A login and signup view
  • A main view listing all of the movies
  • A movie view with description, genre,
    and director of each movie
  • A profile view where users can view
    their information, update information,
    and store their list of favorite movies

Challenges

I had many challenges along the way but I am grateful for the learning experience.
React was a new framework for me and I was still relatively new to JavaScript so everyday
I was running into new situations and problems that I would have to learn-in-the-moment and
trouble-shoot. Facing so many challenges and learning how to troubleshoot has overall made
me a more confident developer.

Duration

So far, this project has been my most favorite, however, it was the most time-consuming. It spanned
a couple of months for me, at least twice as long as any other project during this course.
Fortunately, I had guidance from my mentor for the times I was feeling stuck. Even though it did take quite
a long time to complete, I gained so much from the experience as compared to completing it quickly.
